RazorSettings::RazorSettings ( const QString &  module,
QObject *  parent = 0 
) [explicit]

Constructs a RazorSettings object for accessing settings of the module called module, and with parent parent. RazorSettings can be accessed using the standard interface provided by QSettings, but it also provides some convenience functions and signals.

Parameters:
modulea base name of the config file. It's a name without the extension. For example: if you want to open settings for panel create it as RazorSettings("panel"). The function will create all parent directories necessary to create the file.
parentIt's no need to delete this class manually if it's set.
