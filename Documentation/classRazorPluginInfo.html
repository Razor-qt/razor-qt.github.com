Detailed Description

Every plugin needs a .desktop file that describes it. The basename of this file must be same as the basename of the plugin library.

razorpanel_clock2.desktop file

[Desktop Entry] Type=Service ServiceTypes=RazorPanel/Plugin Name=Clock Comment=Clock and calendar

RazorPluginInfo class gives the interface for reading the values from the plugin .desktop file. This is a pure virtual class, you must implement libraryDir(), translationDir(), and instance() methods.
