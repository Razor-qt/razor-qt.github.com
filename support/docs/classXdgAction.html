Detailed Description

The XdgAction class provides an QAction object based on XdgDesktopFile.

The following properties of the action are set based on the desktopFile. Text - XdgDesktopFile localizeValue("Name") Icon - XdgDesktopFile icon() ToolTip - XdgDesktopFile localizeValue("Comment")

Internally this function will create a copy of the desktopFile, so you can delete original XdgDesktopFile object.

When an action is activated by the user; for example, when the user clicks a menu option, toolbar button or when trigger() was called, XdgAction start the application defined in XdgDesktopFile.

See also:
XdgDesktopFile::startDetached.
