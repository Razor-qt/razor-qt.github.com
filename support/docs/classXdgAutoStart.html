Detailed Description

The XdgAutoStart class implements the "Desktop Application Autostart Specification" from freedesktop.org. This specification defines a method for automatically starting applications during the startup of a desktop environment and after mounting a removable medium. Now we impliment only startup.

See also:
http://standards.freedesktop.org/autostart-spec/autostart-spec-latest.html

XdgDesktopFileList XdgAutoStart::desktopFileList ( bool  excludeHidden = true) [static]

Returns a list of XdgDesktopFile objects for all the .desktop files in the Autostart directories When the .desktop file has the Hidden key set to true, the .desktop file must be ignored. But you can change this behavior by setting excludeHidden to false.
