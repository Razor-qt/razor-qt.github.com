bool RazorAppSwitcher::AppSwitcher::handleEvent ( XEvent *  e)

Catch X11 events to track changes in window orders. Windows last usages is recorded in m_orderedWindows list. Then it is merged with all windows to create logical order of applications in widget (last used first... later used next) in handleApps()
